# curepo.dev

curepo is a curated discovery site for interesting GitHub repositories. It is built with Next.js and provides a lightweight showcase of open-source projects that are worth star-gazing, filtering, and exploring.

## What this site does

- Displays curated GitHub repos with metadata like stars, topics, and descriptions.
- Helps developers discover repositories that match modern tooling, languages, and active maintenance.
- Links visitors directly to third-party GitHub projects rather than hosting code.
- Includes landing, about, and terms pages for a production-ready app structure.
